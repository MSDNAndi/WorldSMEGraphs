## 1. Anatomy & Pathophysiology {#anatomy}

### Abdominal Aortic Branch Vessels - Type 2 Sources

```
        Celiac Trunk (T12)
              │
    Superior Mesenteric (SMA, L1)  
              │
    Renal Arteries (L1-L2) ●─────●
              │
    ═══════════════════
    ║  AORTA  ││││││  ║
    ═══════════════════
         Lumbar Arteries ●●●● ◄── Type 2 Source (4 pairs, L1-L4)
              │
    Inferior Mesenteric (IMA, L3) ◄── Type 2 Source (most common)
              │
         Median Sacral ●  ◄── Type 2 Source
              │
        Common Iliacs /  \
```

### Type 2 Endoleak Mechanism

**NORMAL POST-EVAR** (No leak):
```
    ╔═══════════════════╗
    ║   ENDOGRAFT       ║ ◄── Stent graft excludes aneurysm
    ║                   ║
    ║  [Thrombosed sac] ║ ◄── No blood flow in sac
    ║   IMA ✕ closed    ║
    ║   Lumbar ✕ closed ║
    ╚═══════════════════╝
Result: Sac shrinks, rupture risk eliminated
```

**TYPE 2 ENDOLEAK** (Retrograde flow):
```
    SMA → Collaterals → IMA
                         ▼
    ╔═══════════════════╗
    ║   ENDOGRAFT       ║
    ║                   ║
    ║  IMA ──▶ [SAC]    ║ ◄── Blood flows INTO sac
    ║      Pressurized  ║
    ║   Lumbar ──▶ [SAC]║ ◄── Multiple sources (Type 2b)
    ╚═══════════════════╝
Result: Sac remains pressurized, may expand
```

## 3. Treatment Options {#treatment}

### Decision Algorithm

```
Type 2 Detected
      │
      ▼
 Measure Sac
      │
  ┌───┴───┐
  │       │
Stable   Growing
  │       │
  ▼       ▼
OBSERVE  TREAT
 (6-12mo) (Embolize)
```

### Embolization Techniques

**TRANSARTERIAL** (Via SMA collaterals):
```
Femoral Access → SMA → Collaterals → IMA
                                      ▼
    ╔═══════════════╗
    ║  IMA ──●●● SAC║ ◄── Coils block flow
    ╚═══════════════╝
Success: 70-80% | Recurrence: 20-30%
```

**TRANSLUMBAR** (Direct needle):
```
Posterior skin puncture → Through back muscles → Into sac
    ╔═══════════════╗
    ║   SAC ◄───●── Needle from back
    ║    Fill with  ║
    ║    embolic    ║
    ╚═══════════════╝
Success: 80-90% | Used when transarterial fails
```

## Clinical Pearls

```
✓ Most Type 2 endoleaks are BENIGN (30-50% resolve spontaneously)
✓ Delayed CTA (60-90s) is CRITICAL for detection
✓ Sac BEHAVIOR (not just presence) determines treatment
✓ IMA is most common source
✓ Type 2b (multiple vessels) harder to treat than 2a
✓ Growth >5-10mm/year warrants treatment
```
