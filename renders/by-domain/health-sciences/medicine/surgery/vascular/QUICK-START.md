## For Educators

### Teaching with Comics

**Before Class**:
1. Review the story view to understand the narrative
2. Check learning objectives in the README.md file
3. Consider which viewing format fits your teaching style

**During Class**:
- **Grid view** for discussing multiple panels
- **Story view** for following patient journey
- **Pictures only** for image recognition drills

**After Class**:
- Share links for student review
- Use as reference for clinical cases

### Target Audiences
- Medical students (years 3-4)
- Surgical residents
- Vascular surgery fellows
- Nurse practitioners
- Physician assistants

## Troubleshooting

### Images Not Loading?
- Check you're viewing the markdown file (not raw text)
- Verify images exist in `comic/panels-gpt/` folder
- Try refreshing the page

### File Not Found?
- Use [INDEX.md](INDEX.md) for verified links
- Check [REFACTORING-SUMMARY.md](REFACTORING-SUMMARY.md) for structure details

### Need Different Format?
- All three formats are available for each comic
- Try the grid view if story view is too long
- Use pictures-only for quick review
